# Digital Humanities Citation Network

A citation network visualization of my PhD reading list, mapping connections between primary authors and cited works.

## View the Interactive Network

👉 [Open Interactive Visualization](https://lite.gephi.org/#/graph?url=https://gist.githubusercontent.com/relianceenwerem/512de22387c55fb2874898c29c26b38e/raw/unified.json)

## About

This network maps citation relationships across key Digital Humanities texts, including works by Lauren Klein, Donna Haraway, N. Katherine Hayles, Edward Said, and others.

- **427 nodes** — primary and cited authors
- **Edge weight** — citation frequency
- **Node color** — node type (Primary Author / Cited Author)
